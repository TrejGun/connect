3rd party API's
===============

All 3rd party API's are mocked up by default
You have to enable them by passing XXX_API=true to command line

```bash
XXX_API=true npm run start
```

## Config

* AWS_S3_BUCKET
* AWS_S3_URL
* AWS_S3_REGION
* AWS_ACCESS_KEY_ID
* AWS_SECRET_ACCESS_KEY
* GOOGLE_SERVICE_EMAIL
* GOOGLE_SERVICE_KEYFILE
* STRIPE_PUBLIC_KEY
* STRIPE_PRIVATE_KEY
* MAILGUN_APIKEY
* MAILGUN_DOMAIN
* TWILIO_ACCOUNT_SID
* TWILIO_AUTH_TOKEN
* TWILIO_NUMBER
